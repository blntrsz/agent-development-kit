# Agent slice plan

## Goal
Implement `packages/core/src/agent/` as a vertical slice for:
- loading `.adk/agents/*.ts`
- validating agent definitions
- resolving explicit/default agents
- resolving local/remote agent secrets

Do not couple agent rules to runtime internals. Runtime should consume this slice later.

## Findings
- Current source is mostly empty. `packages/core/src/` only has `sandbox/` and `packages/core/src/sandbox/index.ts` is empty.
- Agent behavior is specified in:
  - `specs/domain/agent/spec.md`
  - `specs/domain/agent/features/define-agent.feature`
  - `specs/domain/agent/features/resolve-default-agent.feature`
  - `specs/domain/agent/features/resolve-agent-secret.feature`
  - `specs/adk-interface.md`
- Repo architecture expects slices under `packages/core/src/<slice>/{index.ts,domain,use-case,port,adapter}`.
- Core package conventions: Effect-native APIs, `Layer` wiring, `Schema` models, tagged errors.

## Recommended approach
### 1. Slice layout
Create:
- `packages/core/src/agent/index.ts`
- `packages/core/src/agent/domain/schema.ts`
- `packages/core/src/agent/domain/model.ts`
- `packages/core/src/agent/domain/errors.ts`
- `packages/core/src/agent/use-case/load-agents.ts`
- `packages/core/src/agent/use-case/resolve-agent.ts`
- `packages/core/src/agent/use-case/resolve-default-agent.ts`
- `packages/core/src/agent/use-case/resolve-agent-secret.ts`
- `packages/core/src/agent/port/agent-module-loader.ts`
- `packages/core/src/agent/port/environment.ts`
- `packages/core/src/agent/port/secret-store.ts`
- `packages/core/src/agent/adapter/bun-agent-module-loader.ts`
- `packages/core/src/agent/adapter/local-environment.ts`
- `packages/core/src/agent/adapter/in-memory.ts`
- `packages/core/src/agent/adapter/layer.ts`

### 2. Domain model
Define schema-backed models for:
- `SecretRef`
- `AgentConfig`
- `RegisteredAgent`
- `AgentRegistry`
- `ExecutionMode = "local" | "remote"`

Normalize secret refs early to one internal shape:
- `{ localEnv?: string; secretsManager?: string }`

Keep `defineAgent(config)` as a typed identity helper. Enforce real validation at load time.

### 3. Errors
Add tagged errors for:
- invalid agent module
- duplicate agent name
- file name and `name` mismatch
- agent not found
- missing default agent
- invalid secret ref
- secret not found

### 4. Use-cases
Implement:
- `loadAgents(projectRoot)`
  - list `.adk/agents/*.ts`
  - reject nested paths
  - import each module
  - require default export from `defineAgent(...)` shape
  - enforce file name matches `config.name`
  - reject duplicate names
- `resolveAgent(registry, explicitAgent?)`
  - use explicit agent when provided
  - else use default agent
- `resolveDefaultAgent(registry)`
  - require agent loaded from `.adk/agents/default.ts`
- `resolveAgentSecret(agent, mode)`
  - local: read env
  - remote: read secret store
  - never create remote secrets

### 5. Ports and adapters
Ports:
- `AgentModuleLoader`
- `Environment`
- `SecretStore`

Adapters now:
- Bun loader for `.adk/agents/`
- local env adapter
- in-memory test adapters
- Effect `Layer` composition for live/test wiring

Defer real AWS adapter. First pass keeps remote secret access behind `SecretStore` only.

### 6. Public API
Expose from `packages/core/src/agent/index.ts`:
- `defineAgent`
- exported types/schemas needed by callers
- core use-case entrypoints
- tagged errors
- live layer factory if needed

Keep `defineAgent` slice-only in first pass. Keep ports/adapters internal unless tests or runtime wiring need them.

### 7. Tests
Add Bun tests for:
- load agent from `.adk/agents/triage-bot.ts`
- reject nested agent files
- reject missing/invalid default export
- reject file-name and `name` mismatch
- reject duplicate names
- resolve explicit agent
- resolve default agent
- fail when implicit default missing
- resolve local string secret from env
- resolve object secret ref local vs remote
- fail when secret object has no fields
- fail when remote secret missing and prove no create path exists

Prefer colocated tests under `packages/core/src/agent/use-case/*.test.ts` unless another convention is chosen.
